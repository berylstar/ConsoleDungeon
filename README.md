# ConsoleDungeon

> 콘솔 던전에 오신 것을 환영합니다. 콘솔에서 펼쳐지는 던전에 도전하세요 !

# 구현 기능
- 말하는 것 같은 문자열 출력 + 문자열 색상 변경
- 게임 시작 화면
- 플레이어 스탯 - 공격력, 방어력, 체력, 스피드
- 인벤토리 - 장비 장착 / 해제, 그에 따른 효과 적용, 타입별 장비 장착 개선
- 상점 - 장비 구매, 장비 판매
- 던전 난이도 시스템
- 전투 시스템 - 공격, 방어, 집중
- 레벨 업 시스템

# 기능 설명

`Program.Talk()` 메소드를 통해 문자열을 한 글자씩 출력하며 말하는 것 같은 기능을 추가했습니다. `Program.ColorWrite()` 메소드를 통해 문자열의 색상을 다르게 출력하는 기능을 추가했습니다.

게임 관련 메소드는 `GameController` 클래스에 넣었습니다. 

먼저 게임 시작시 `GameConroller.GameStart()` 메소드를 실행시켜 게임을 시작하고 플레이어의 이름과 직업을 선택할 수 있습니다. 직업에 따라 능력치를 다르게 설정했습니다.

플레이어의 입력을 받을 때는 `GameController.GetInput()` 메소드를 통해 올바른 입력값만 받을 수 있도록 구현했습니다.

다음 `GameConroller.VillageEnterance()` 메소드를 통해 필수 구현 요소를 구현했습니다. 선택지는 상태 보기, 인벤토리, 상점 - 구매, 상점 - 판매, 던전 입장, 게임 종료입니다.

플레이어 스탯을 확인 할 때는 GameController 클래스에 미리 Player 클래스를 생성시키고 필드에 저장시켜 이 클래스를 참조해 값을 가져오도록 했습니다.

인벤토리에서는 획득한 장비와 그에 대한 설명이 나오게 했습니다. 인벤토리의 레이아웃을 고정하기 위해 `string.PadRight()` 메소드를 활용했습니다. 또한 같은 타입의 장비는 하나만 장착가능 합니다. 이는 `Equip.Switch()` 메소드에서 구현해 같은 함수로 장착 또는 해제를 할 수 있도록 구현했습니다. 장착 또는 해제시에 장비의 효과를 적용시키거나 해제하기 위해 `Player.EquipEffect()` 메소드를 구현했습니다.

상점 기능은 `GameController.ShowShopForBuy()` 메소드에서 상점에서 원하는 장비를 구매하는 기능을, 상점에서 판매하는 장비들은 `Shop` 클래스에 넣었습니다. 먼저 생성자에서 전체 장비를 `Shop.AllEquips` 에 넣고 `GameController.SetShop()` 메소드를 통해 랜덤한 5개의 장비만 판매하도록 구현했습니다.

`GameController.ShowShopForSell()` 메소드에서 인벤토리의 장비를 상점에 판매하는 기능을 구현했습니다.

이후 `GameController.InDungeon()` 메소드로 던전에 입장할 수 있습니다. `Dungeon` 클래스에서 던전을 관리하도록 구현했습니다. 던전은 Easy -> Normal -> Hard -> Boss 순서의 난이도로 진행합니다. 이전 난이도의 던전을 클리어해야 다음 던전에 도전 가능 합니다. `Dungeon.SetMonsters()` 메소드에서 난이도에 따라 몬스터의 수를 다르게 설정했습니다. 던전에 입장하면 전투를 시작하기 전 상태와 인벤토리를 확인한 후에 전투를 실행할 수 있습니다.

전투는 `GameController.ShowBattle()` 메소드로 구현했습니다. 랜덤한 능력치를 가진 몬스터와의 전투가 시작되며 몬스터는 `Dungeon.Monsters` 리스트에 있는 몬스터와 도전하게 되며 몬스터를 처리하면 리스트에서 첫 번째 원소를 제거하는 방식으로 리스트의 원소 갯수로 던전을 클리어 했는지 판단합니다.

전투는 3가지 선택지가 있습니다. 공격, 방어, 집중. 전투 선택지는 `GameController.CharacterAction()` 메소드로 구현했습니다. 사용하는 매개변수에 인터페이스를 넣어 플레이어나 몬스터가 모두 사용할 수 있도록 구현했습니다. 또한 플레이어나 몬스터 중 스피드가 높은 쪽이 먼저 액션을 취할 수 있습니다.

1. 공격 : 상대를 현재 공격력만큼 공격합니다. 집중 카운트에 비례해 추가 데미지가 들어갑니다.
2. 방어 : 방어 태세에 들어가며 상대의 공격을 방어할 수 있습니다. 하지만 연속해서 방어하기는 불가능하니 적절하게 사용해야 합니다.
3. 집중 : 집중 카운트를 한 개 증가시킵니다.

전투 중에 발생하는 메세지는 `Program.script` 큐에서 관리하고 있으며 큐에 메세지를 Enqueue 하고 다음 전투 턴에서 모든 메세지를 Dequeue하는 방식으로 전투 메세지가 나타나도록 구현했습니다.

몬스터를 처치하면 경험치와 골드를 보상으로 받을 수 있습니다. 경험치는 `Player.GetExperience()` 메소드를 통해 레벨 업을 할 수 있습니다.

던전을 클리어하면 다음 난이도의 던전이 세팅되고, 상점이 새로고침 된 이후 다시 마을로 돌아와 반복하게 됩니다.

전투 도중 플레이어가 죽으면 `GameController.GameFail()` 메소드를 실행시켜 게임이 실패하고, 마지막 보스를 처치하면 `GameController.GameWin()` 메소드를 실행시켜 게임에서 승리합니다.
